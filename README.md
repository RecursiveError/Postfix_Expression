# Postfix Expression
Pequeno teste de conversão de infix para postfix

**IMPORTANTE** 
Esse programa é uma implementação simples, não contem: 
> - tratamento de erros 
> - operações com raiz e potenciação 
> - operações implicitas a(b) ou ab == a*b 


build and Run
```properties
make && ./build/main
```
